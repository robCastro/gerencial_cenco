Realización de ETL

Atención, el proceso de actualización de datos presenta una gran carga tanto para el sistema gerencial como para el administrativo, se recomienda realizarlo en las horas cuyo uso sea menor.

El ETL consiste en la extracción de datos del sistema transaccional e insertarlos en la base de datos gerencial, este proceso supone una gran carga para ambas bases de datos, se recomienda que se realice entre 11 pm y 5 am. Dependiendo de la cantidad de datos, podría llegar a tomar hasta 5 minutos.
